# Interactive Dashboard on Online Course Completion Rates

## Overview
This project is an interactive Tableau dashboard developed to analyze online course completion trends, learner engagement, and enrollment patterns across different learning platforms.

## Features
- Course-wise completion analysis
- Platform-wise enrollment trends
- Completion rate visualization
- Learner engagement insights
- Dashboard filters and KPI metrics

## Tools Used
- Tableau
- Excel
- Data Visualization
- Statistical Analysis

## Key Insights
- Identified enrollment trends across multiple online learning platforms
- Analyzed learner completion behavior and engagement patterns
- Visualized completion metrics using interactive dashboards and charts
